# Outputs Folder

Temporary evaluation reports and results are saved here during benchmarking and validation runs.

Examples:
- `ablation_results.json` - Comparison of chunking strategies
- `retrieval_optimization_results.json` - Hybrid weight tuning results
- `final_validation_report.json` - 15-query benchmark performance

**Note**: These files are excluded from git and regenerated locally or in Spaces as needed.
